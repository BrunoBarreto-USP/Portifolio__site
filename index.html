Projetos

Redes Neurais para Regressão de Parâmetros Estelares (05/2025)

Desenvolvi um modelo de aprendizado profundo para estimar parâmetros fundamentais de estrelas — Temperatura Efetiva (Teff), Metalicidade ([Fe/H]) e Gravidade Superficial (logg) — a partir de espectros 1D.

Ver no GitHub

Rede Neural Convolucional (CNN) para Classificação de Galáxias (04/2025)

Criei um modelo para categorizar imagens de galáxias do conjunto Galaxy 10 SDSS em 9 classes distintas, explorando a interseção entre aprendizado profundo e astronomia.

Ver no GitHub

Classificação de Espectros de Objetos Astronômicos (04/2025)

Desenvolvi um projeto de Machine Learning para classificar galáxias, quasares e estrelas utilizando Redes Neurais e espectros do SDSS.

Ver no GitHub

Uso de IA/ML na Astrofísica VS Recursos Computacionais (07/2025)

Desenvolvi um script em Python para coletar, processar e analisar mais de três décadas de artigos de astrofísica do arXiv, quantificando a correlação entre o crescimento do poder das GPUs e a adoção de técnicas de IA.
